инструкция

 1. Visual Studio 2026.

2. На стартовом окне выбрать Клонировать репозиторий.

3. Вставить ссылку на GitHub-репозиторий проекта в поле URL и выбрать папку, в которую он будет загружен.

4. Нажать Клонировать.

5. После загрузки открыть решение проекта (.sln), если оно не открылось автоматически.

6. Дождаться восстановления зависимостей NuGet.
Если пакеты не восстановились автоматически, в верхнем меню выбрать:
Сервис → Диспетчер пакетов NuGet → Восстановить пакеты.

7. Открыть Package Manager Console и выполнить команду для применения миграций: Update-Database

(опционально) Если база данных пустая, добавить тестовые данные вручную через SQLite DB Browser или другим предусмотренным способом.

8. Выбрать профиль запуска проекта и нажать F5 или кнопку Запуск.

9. После сборки Visual Studio откроет приложение в браузере по локальному адресу
